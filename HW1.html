Introduction and Review

Functionality in R comes from various packages that do various things. Loading packages is an important first step in the vast majority of cases.

You can load packages using the require command. One commonly-used package is mosaic. Make sure to include the message=FALSE option in the chunk that loads mosaic. This will suppress unwanted messages and keep your Markdown document neat and clean.

require(mosaic)

Note that R is case-sensitive. That means capitalization counts. Like any programming language, spelling counts too. If you are unable to load the mosaic package using the preceding command, make sure that it is installed in your librarys. Click on the Packages tab in the lower right-hand corner of RStudio, and make sure you see mosaic listed. If not, click on Install.

Univariate Analysis

We will use the formula interface whenever possible. This means using the ~ (tilde) operator. To get a basic summary of a variable, use favstats().

favstats(~height, data = Galton)
##  min Q1 median   Q3 max     mean       sd   n missing
##   56 64   66.5 69.7  79 66.76069 3.582918 898       0

To specifically retrieve the mean or standard deviation, use the respective commands.

mean(~height, data = Galton)
## [1] 66.76069
sd(~height, data = Galton)
## [1] 3.582918

You can store these values to new variables if you want to use them later on.

Note that while both the assignment operator (<-) and the equal sign (=) do the same thing, we prefer the former for clarity.

mu <- mean(~height, data = Galton)
mu
## [1] 66.76069

For a graphical description of the distribution of a single variable, we can create a histogram or a density plot.

histogram(~height, data = Galton)

densityplot(~height, data = Galton)

Bivariate Analysis

For assessing the relationship bewteen two variables, we can draw a scatterplot.

xyplot(height ~ mother, data = Galton)

Note that there are many graphical options that we can add to this plot. Here is the same basic plot, but jazzed up a bit. You should experiment with graphical options! If you dont know what is available, hit the tab key inside the function call. Or type help(xyplot).

xyplot(height ~ mother, groups = sex, data = Galton, 
       type=c("p", "r", "smooth"), 
       auto.key = list(columns = 2), 
       pch = 19, alpha = 0.3, cex=1.5, lwd = 3, 
       main="Relationship between Adult Height and Mother's Height", 
       ylab="Height of Adult (inches)", 
       xlab="Height of Adult's Mother (inches)")

Note that in this case we have conditioned on a third variable sex, by adding the groups argument.

For a numerical assessment of the linear relationship between two quantitative variables, we can compute their correlation coefficient.

cor(height ~ mother, data = Galton)
## [1] 0.2016549

For relating a quantitative variable to a categorical variable, we can draw a box-and-whisker plot.

bwplot(height ~ sex, data = Galton)

A table can also be useful here.

favstats(height ~ sex, data = Galton)
##   sex min   Q1 median   Q3  max     mean       sd   n missing
