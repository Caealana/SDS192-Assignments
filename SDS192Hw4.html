  1. Write a function getFlights() that will determine, for a given airport code, the number of distinct flight numbers (e.g. flight not the number of flights!) that fly into and out of Bradley. Your function should take two arguments: a data frame called data and code, which is the three character airport code. It should return a data.frame with two columns and one row. The first column should be called incoming and give the number of distinct flight numbers that come into Bradley from that airport. The second column should be called outgoing and give the number of distinct flight numbers that come out of Bradley to that airport.

Your output should look like this:

getFlights(data = bdl, code = "DFW")
## incoming outgoing 
##        8       14
#input: data, airport code
getFlights <- function(data, code){
  
  dataOrigin <- data %>%
    filter(origin == code)%>%
    select(flight) %>%
    unique()
  
  dataDest <- data %>%
    filter(dest == code) %>%
    select(flight) %>%
    unique()
  
  incoming = c(nrow(dataOrigin))
  outgoing = c(nrow(dataDest))
  df = data.frame(incoming, outgoing)
  df
  
  
}
